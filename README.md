# 🏗️ Concrete Mix Design Pro Tool

## 📘 Description
This is a web-based concrete mix design application developed using Streamlit.

## 🔧 Features
- Mix design (M20–M40)
- Nominal & Design mix identification
- Material replacement (Fly ash, GGBS, Silica fume, Rice husk ash)
- Cost estimation
- CO₂ analysis
